CTC Technologies (last updated 2019)

About This Project:

The primary purpose of this website is a marketing vehicle to generate sales for CTC Technologies.

  • On the Partners page, each partner is a post in the "Partners" custom post type. Each partner is assigned a category (custom taxonomy). The category is added to each logo as a class, and filtered out with a jQuery script.
  • The navigation menu is a native WordPress menu, styled with CSS and made interactable with a customized jQuery plugin
  • On mobile on the home page, the "solutions" section uses the same basic script as the navigation, and the data is entered with ACF.
  • Case Studies are their own custom post type, and are filtered by custom taxonomies on the page using FacetWP

Heartland Sheds (2018/2019)

About This Project:

This is an e-commerce website using WooCommerce. This was a very complex e-commerce project with a lot of conditional logic needed for each shed. Most of this logic was done by other developers. My role was to supply the presentation layer and code in the ACF fields for each product. I will describe some of the key components that I developed below.

  • On the sheds page, hovering on the shed reveals the different sizes available, as well as reviews. Clicking on a size brings you to the product page with that size selected (accomplished with query strings).
  • When you initially arrive on a product page (Ranch shed as an example), there is a 360 video of the currently selected shed size. You can hover on the icons to the left to view different photos in the gallery.
    • Clicking on the large image opens a lightbox scrolled to the position of that image in the gallery.
  • On mobile, the gallery is simplified to a scrolling gallery with no thumbnails.
  • The comparison tables are created using Tablepress (to make it easier for the client to edit).
  • Clicking on a size on the right shows a different shed. All of the data below the fold also changes to reflect the new shed size. This data is all entered on the product page with ACF.
  • Hovering over the shed size shows the price of that size.
  • Shed sizes are custom product attributes.
  • For marketing/SEO purposes, a page template was created for the shed sizes with content that is edited in the description box on the attribute page in the dashboard. A WordPress query selects the sheds of that size and displays them below the content so that the customer can navigate directly to the product page. A list of shed sizes is also generated and can be viewed here.
  • A Branch Locations page is generated with WP Store Locator, and customized with CSS and a bit of PHP to add some features to each listing (such as the "view branch info" button).
    • Other integrations: affirm, yotpo, avatax, and mailchimp are integrated with this site.
